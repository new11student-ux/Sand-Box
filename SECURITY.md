# Security Policy

## Supported Versions
Only the latest version of the Advanced Cybersecurity Sandbox Platform on the `main` branch is actively supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability within this project (such as sandbox escapes, arbitrary code execution, or credential leaks), please **DO NOT** create a public issue.

Instead, please send an email to `security@sandbox-platform.local` (Replace with the actual researcher email in production).

### What to include
- Description of the vulnerability.
- Steps to reproduce the issue.
- Proof of Concept (PoC) code or screenshots.
- Potential impact (e.g., host compromise, lateral movement).

We will acknowledge receipt of your vulnerability report within 48 hours and strive to provide a patch within 7 days.
